Для работы Selenium Grid необходимо иметь следующее:
1. Установленную среду выполнения Java (JRE) или набор для разработки Java (JDK). Предпочтительнее использовать последнюю версию JDK, но подойдёт любая версия выше Java 11 (я использую eclipse и JDK 17).
2.Установленный браузер или браузеры (я использую Chromium, Edge и Firefox). 
3.Драйверы браузеров (я использую geckodriver, chromedriver и msedgedriver).
4.Скачанный jar-файл Selenium Server (я использую selenium-server-4.25.0.jar).
5. bat-файл для запуска selenium-server-....jar в автономном режиме.

Написание автотеста.
1. Создать файл в eclipse с необходимым Class, имя файла должно совпадать с именем класса.
2. Импортировать необходимые методы.
3. Если необходимо чтобы тесты запускались в определённом порядке. Необходимо указать аннотацию Order
4. Перечисляем driver в WebDriver, которые будем использовать.
5. Пропишем предусловие, в котором создаём драйвер. Т.е. здесь прописываем опции для драйверов и запускаем драйвера. Ещё можем создать переменную.
6. В постусловии закрываем наши драйвера.
7. пишем название теста №1.
	7.1. с помощью драйвера запускаем необходимый сайт
	7.2. с помощью драйвера задаём размер запускаемого браузера
	7.3. с помощью драйвера задаём параметры треугольника
	7.4. с помощью драйвера жмём рассчитать параметры треугольника
	7.5. указываем идентификатор потока, в к-м будет запускаться наш тест 
	7.6. выводим на экран чему будет равен идентификатор потока 
	7.7. сравниваем ожидаемый результат с фактическим
8. пишем название теста №2.
	8.1. с помощью драйвера запускаем необходимый сайт
	8.2. с помощью драйвера задаём размер запускаемого браузера
	8.3. с помощью драйвера сбрасываем параметры треугольника
	8.4. с помощью драйвера задаём параметры треугольника
	8.5. с помощью драйвера жмём рассчитать параметры треугольника
	8.6. указываем идентификатор потока, в к-м будет запускаться наш тест 
	8.7. выводим на экран чему будет равен идентификатор потока 
	8.8. сравниваем ожидаемый результат с фактическим
9. Создать ещё три теста с этого же сайта (из пункта 8), но с другими параметрами треугольника.
10. пишем название теста №6.
	10.1. с помощью драйвера запускаем необходимый сайт
	10.2. с помощью драйвера закрываем всплывающий баннер (о скачивании приложения), чтобы добраться до сайта
	10.3. с помощью драйвера выбираем строку поиска для ввода необходимого
	10.4. с помощью драйвера вводим необходимое слово, для поиска
	10.5. с помощью драйвера отправляем нажатие клавиши ENTER
	10.6.задаём задержку в 3 секунды для корректной отработки запроса
